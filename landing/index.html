Architecture & Mail Flow

Архітектура та Рух Листів

Understanding how mail traffic enters, is filtered, and is securely delivered to endpoints.

Як поштовий трафік потрапляє на сервер, проходить фільтрацію та безпечно зберігається.

01

Inbound Routing

Вхідна Маршрутизація

NFTables routes external traffic securely from WAN ports directly into DMS containers, keeping host safe.

NFTables надійно перенаправляє зовнішній трафік із портів 25, 465, 587, 993 прямо в контейнер.

02

Spam & Virus Filters

Фільтрація Спаму

Rspamd performs multi-metric analysis (SPF, DKIM, DMARC, DNSBL, Bayes) to block malicious mail instantly.

Rspamd аналізує листи за багатьма метриками (SPF, DKIM, DMARC, DNSBL) для миттєвого блокування спаму.

03

SSL/TLS Encryption

SSL/TLS Шифрування

Let's Encrypt certificates are mounted read-only to ensure absolute privacy and fully secure client logins.

Сертифікати Let's Encrypt монтуються в режимі read-only для шифрування підключень клієнтів.

04

Active Protection

Активний Захист

Fail2Ban monitors audit logs and automatically updates firewall tables to drop brute-force botnets.

Fail2Ban сканує системні логи та блокує IP-адреси зловмисників за спроби підбору паролів.

Deployment & Quick Start

Встановлення та Налаштування

Deploy your own mail server cluster in three simple steps.

Запустіть власний поштовий сервер всього за три простих кроки.

1

Clone & Prepare Workspace

Клонування та Підготовка Репозиторію

Clone the mail server repository and switch to the project directory:

Завантажте репозиторій на свій сервер:

git clone https://github.com/weby-homelab/docker-eMailServer.git && cd docker-eMailServer
2

Run Automated Installation

Запуск Автоматичного Встановлення

Use the pre-packaged shell installer to initialize directories, templates, and config files:

Виконайте інсталяційний скрипт для конфігурації папок та середовища:

chmod +x install.sh && ./install.sh
3

Configure DNS Records

Налаштування Записів DNS

To ensure maximum deliverability and SPF/DKIM/DMARC pass, enter your domain below to generate the exact DNS configurations:

Для того, щоб листи не потрапляли в спам, введіть свій домен нижче для генерації точних налаштувань DNS записів:

TypeТип Host/NameХост/Ім'я ValueЗначення
MX @
10 mail.example.com.
TXT @
v=spf1 mx ip4:YOUR_SERVER_IP -all
TXT
mail._domainkey.example.com.
v=DKIM1; k=rsa; p=...
TXT
_dmarc.example.com.
v=DMARC1; p=reject; pct=100; rua=mailto:dmarc@example.com

Admin Dashboards & Monitoring

Панелі Управління та Моніторинг

Monitor mail queues, statistics, spam ratios, and resources in real time.

Відстежуйте черги листів, статистику спаму та навантаження в реальному часі.

Rspamd Web UI Admin Panel

Rspamd Web UI Панель

Rspamd features an advanced dynamic interface to view spam scores, learn filters, and scan headers. For security, it binds only to localhost.

Rspamd надає потужну динамічну панель для перегляду оцінок спаму, навчання фільтра та сканування заголовків. З метою безпеки панель прив'язана лише до localhost.

ssh -N -L 11334:127.0.0.1:11334 root@mail.example.com
Then navigate to: http://localhost:11334 Після цього відкрийте: http://localhost:11334

Prometheus & Grafana Integration

Інтеграція Prometheus & Grafana

Exposes Postfix metrics via postfix-exporter on port 9154. Allows instant scraping and connection to dynamic dashboard suites.

Експортує метрики пошти через postfix-exporter на порту 9154. Дозволяє миттєво підключити Grafana для візуалізації графіків трафіку.

Hardening & Security Features

Захист та Безпека

The stack runs maximum security configurations out-of-the-box.

Конфігурації стеку налаштовані на максимальну безпеку з коробки.

Modern TLS 1.3

Сучасний TLS 1.3

Enforces modern encryption layers only. Weak ciphers are disabled to prevent SSL handshake leaks and downgrade attacks.

Використовує лише сучасні шифри. Слабкі алгоритми примусово вимкнено для захисту з'єднань від компрометації.

Fail2Ban Active Defense

Fail2Ban Захист

Fail2Ban monitors log outputs in real time, automatically dropping IPs attempting SMTP brute-force or dictionary sweeps.

Автоматично сканує поштові логи та миттєво блокує ботнети при спробах підбору паролів до скриньок.

rDNS / HELO Mandate

Вимога rDNS / HELO

Strict checks ensure EHLO/HELO names align with server PTR pointer records, boosting sender reputation to 10/10.

Суворе узгодження вихідного EHLO/HELO хостнейма із PTR записом IP поштового сервера забезпечує рейтинг відправки 10/10.
